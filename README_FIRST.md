# Kit para o Codex — Jumpstyle Brasil

## Antes de executar

O endereço solicitado, `https://jumpstylebrasil.github.io/`, não é produzido por um repositório `Mreaggle/jumpstylebrasil`.

Para a URL exata, crie ou disponibilize o owner GitHub `jumpstylebrasil` e autorize o agente nessa conta/organização. O repositório correto será:

```text
jumpstylebrasil/jumpstylebrasil.github.io
```

Caso mantenha o owner `Mreaggle` e use o repo `jumpstylebrasil`, o endereço será:

```text
https://mreaggle.github.io/jumpstylebrasil/
```

## Uso

1. Extraia este ZIP em uma pasta de trabalho.
2. Abra a pasta no Codex.
3. Garanta que `gh auth status` esteja autenticado na conta correta.
4. Cole o conteúdo de `CODEX_ONE_SHOT_PROMPT.md`.
5. Não envie PAT, token ou senha dentro do prompt.

## Arquivos

- `CODEX_ONE_SHOT_PROMPT.md`: prompt principal.
- `MIGRATION_AUDIT.md`: diagnóstico e inventário.
- `inputs/`: fontes originais.
- `reference-renders/`: imagens das 9 páginas para inspeção visual.
- `manifests/`: conteúdo, links e fontes do JUN em formato estruturado.
