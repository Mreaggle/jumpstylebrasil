# Alteracoes editoriais

Nao foram feitas correcoes ortograficas silenciosas no texto original preservado.

Os textos de interface foram revisados com acentuação correta e linguagem voltada ao público final. O conteúdo de origem permanece integral em `src/data/original-content.json`, sem ser exibido nas rotas públicas.

Blocos de evidência visual, textos integrais de origem e referências a migração, snapshots, build ou auditoria foram removidos do HTML público. Esses elementos continuam disponíveis apenas nos arquivos internos do projeto.

A rota editorial `/fireborn-squad/` foi adicionada fora das nove páginas migradas. O cadastro numerado fornecido pela equipe foi tratado como fonte canônica para situação, grau e títulos; a seção resumida "por grau" não foi usada para sobrescrever esse cadastro porque contém divergências de Praetor.
