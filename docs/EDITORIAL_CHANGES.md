# Alteracoes editoriais

Nao foram feitas correcoes ortograficas silenciosas no texto original preservado.

As paginas novas usam chamadas curtas e sem acentos em alguns textos de interface para manter compatibilidade simples de codigo, mas o conteudo original de origem permanece integral em `src/data/original-content.json` e nos blocos "Texto original preservado" de cada rota.

A rota editorial `/fireborn-squad/` foi adicionada fora das nove páginas migradas. O cadastro numerado fornecido pela equipe foi tratado como fonte canônica para situação, grau e títulos; a seção resumida "por grau" não foi usada para sobrescrever esse cadastro porque contém divergências de Praetor. A página distingue explicitamente registro da equipe e fatos históricos documentados na JUN.
